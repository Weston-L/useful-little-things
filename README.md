<li>
  <a href=https://github.com/Weston-L/useful-little-things/blob/main/ShadowMaker/ShadowMaker.py>
      ShadowMaker.py - This little python script finds all the image files in a folder, puts a shadow behind the image, then saves the png image and renames it to the name of the folder. The settings are customizable, so you can give your image as dark and large/small a shadow as you want.
</li><p>
<li>
  <a href=https://github.com/Weston-L/useful-little-things/blob/main/ShadowMaker/ShadowMaker-deletes-white-threshold.py>
      ShadowMaker-deletes-white-threshold.py - This little python script is very similar to ShadowMaker.py, but it erases everything from the image that is too white (you can set the white threshold) and puts a shadow behind the image. It also automatically renames the images to the folder name + numbers.
</li><p>
